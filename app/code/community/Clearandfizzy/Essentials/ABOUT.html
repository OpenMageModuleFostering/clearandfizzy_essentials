Sometimes you just need to output raw content without all the wrappers Magento places around your Page Content or Static Blocks. But still Manage this within Magento.

Key Features

* Manage - Keep all you raw content within Magento using it's custom management screen.
* Content-types - Set your own content type, want to output XML data -- no problem!
* CMS Pages - Your raw content is included via a Magento Widget using the stand Magento Pages screens.
* CMS Pages - By using Magento's CMS Pages you have access to Magentos MultiStore URLs and everything that goes with.

Support and Guarantee

This is a FREE plugin and comes with limited email support.

Please note that this extension is provided as is. It is recommended to always backup your installation prior to use.
